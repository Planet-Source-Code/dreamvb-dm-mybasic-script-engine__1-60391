    Data Types Top
    In programming languages we use data types that are known as variables. That is space in the computer memory to store various data. Out language at the moment can support 3 types. See below:

    1. Integer
    2. String
    3. Varient

    The     Integer Datatype is used to store only numeric data eg as 0..9
    The     String Datatype can be used to store string data. eg "this is a string" or "125" incolosed in quotes "

    The Variant Data type is capable of storing any sort of data include String, Integer and floating point numbers. However in our language each data type is in fact of type Variant. Though this is hidden from the user. Though the user of conversion functions allow us to use such data types as Integer and String to obtain the same result.
    Quick Guide Keywords Top

    Keyword DIM: <VarName> [VarDatType]

    The Dim keyword is used to allow you to create a variable in the computers memory to store data. Note [VarDatType] here you should place here your data type. See data types above. If [VarDatType] is left empty your variable is assumed to be of type Variant.

    Examples:
    DIM Name As String
    DIM Age As Integer
    DIM Other <-- Varian

    Keyword REM:
    REM keyword is used to add comments to your program. To help you remember what the code is about.

    Example:
    REM This prints the value of 5 and 6
    PRINT 5+6
    REM Add other comments here


    Keyword CLS:
    CLS is used to clear the console window of it's data

    Example:
    PRINT "Some Text"
    CLS
    PRINT "In this example you never see the top line"


    Keyword PRINT: Expression
    Prints data to the console.

    Examples:
    PRINT "HI String"
    PRINT 5+4
    PRINT A+B
    PRINT "Hello String " & (5.6+5.6)


    Keyword INPUT: [Prompt], Variable
    Takes input from the console. And stores the result in to Variable.
    Prompt Allows you to display a message on the console.

    Example:
    DIM NAME AS STRING
    INPUT "What is Your Name",Name
    PRINT "HELLO " & Name


    Keyword BEEP:
    Send the Bell control chr 7 to the console to make the pc speaker Beep.

    Example:
    BEEP


    Keyword PAUSE:
    Pause is used to pause the console until a key is pressed.
    This comes in handy as the console can flash of quickly.

    Example:
    PRINT "Pause"
    PAUSE


    Keyword LET: <Variable>=<Expression>
    Allows the assignment of a expression to a variable.

    Example:
    Dim A
    Dim B as Integer
    DIM C
    LET A = "Hello Number: "
    LET B = 10
    LET C = A & B
    PRINT C

    Keyword LOCATE: <Expression>,<Expression>
    The locate keyword takes in two expressions and positions the cursor in the console based of the values.

    Example:
    DIM A
    DIM B
    LET A = 20
    LET B = 1
    LOCATE 10,10
    PRINT "A"
    LOCATE A,B
    PRINT "B"

    Keyword END:
    Terminates the execution of a program and returns to the operating system.

    Example:
    PRINT "Hello"
    PAUSE
    END
    PRINT "You will not see me because the program has ended"
